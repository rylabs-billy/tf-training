We created Linodes

These Linodes are deployed via Terraform in the Sydney data center.

The Linode API token, root passowrd, ssh key, and amount were all defined as variables.

We secured them with a VLAN and Cloud Firewall

The VLAN is confgured on eth1 interface. The addresses are 10.0.0.X/24"

The Cloud Firewall is attached to all Linodes from the "linode_instance" "web" resource

We utilized a NodeBalancer for HA

This took configuration of three Terraform resources:

"linode_nodebalancer"

"linode_nodebalancer_config"

"linode_nodebalancer_node"
